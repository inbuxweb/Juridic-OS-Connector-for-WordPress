**El plugin oficial de Juridic-OS para integración de formularios de contacto con sistemas de gestión legal.**

== Description ==

Juridic-OS Connector permite conectar los formularios de contacto de tu sitio web directamente con tu sistema de gestión legal, automatizando la captura y procesamiento de leads.

La integración es **simple y no requiere conocimientos técnicos avanzados**, permitiéndote comenzar a gestionar tus contactos de inmediato.

== Características de Integración ==

Juridic-OS Connector soporta:
* Contact Form 7
* Otros formularios populares de WordPress
* Sincronización automática de datos
* Configuración rápida desde el panel de administración

== ¿Por qué elegir Juridic-OS Connector? ==

* Sin tarifas de configuración
* Sincronización instantánea de formularios
* Gestión centralizada de leads
* Integración directa con tu sistema legal Juridic-OS
* Seguridad en la transferencia de datos

== Instalación ==

= Instalación Automática =

1. Ingresa al panel de WordPress
2. Ve a Plugins > Añadir Nuevo
3. Busca "Juridic-OS Connector"
4. Haz clic en Instalar y luego Activar

= Instalación Manual =

1. Descarga el archivo ZIP del plugin
2. En WordPress, ve a Plugins > Cargar
3. Selecciona el archivo ZIP
4. Instala y activa el plugin

== Configuración ==

1. Ingresa a tu cuenta de Juridic-OS
2. Obtén tus credenciales de API
3. Configura el plugin con tus claves
4. Selecciona los formularios a conectar

== Preguntas Frecuentes ==

= ¿Funciona con Contact Form 7? =
Sí, es totalmente compatible con Contact Form 7.

= ¿Necesito una cuenta de Juridic-OS? =
Sí, necesitas una cuenta activa para usar este plugin.

= ¿Dónde almacena los datos? =
Los datos se envían directamente a Juridic-OS, sin almacenarse en WordPress.
